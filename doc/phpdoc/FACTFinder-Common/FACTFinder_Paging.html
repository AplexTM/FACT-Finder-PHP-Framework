Class: FACTFinder_Paging

Source Location: /Paging.php

Class Overview




Class for creating pagelinks on a search result page. by iterating over an paging object, you will get FACTFinder_Item  objects which represent the link for each page, beginning from "getFirtPageNumberShown()" to "getLastPageNumberShown()".
